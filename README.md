# Dotfiles

> Dotfiles for my arch linux build

## Configs:
- i3-gaps (WM)
- i3blocks (Status bar)
- kitty (Terminal emulator)
- rofi (Window switcher)
- picom (Compositor)
- lightdm (Display manager)
- zsh (with oh-my-zsh) (Shell)
- doom emacs (Ttext editor)
